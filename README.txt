█ BACKUP YOUR DATA

  From within the app:
  1. Click Dashboard
  2. Go to Settings → Database Operations
  3. Click "Export Database"
  4. Save the .db file somewhere safe
  5. Keep copies as backup


█ RESTORE FROM BACKUP

  1. Go to Settings → Database Operations
  2. Click "Import Database"
  3. Select your backup .db file
  4. Click Restore
